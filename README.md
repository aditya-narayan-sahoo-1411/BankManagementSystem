
# 🏦 Bank Management System

A **Bank Management System** developed using **Python** and **Streamlit** that allows users to perform essential banking operations through a simple and interactive web interface. The application stores account information in a JSON file, making it lightweight and easy to use without requiring a database.

---

## 📌 Features

* 🆕 Create a New Bank Account
* 💰 Deposit Money
* 💸 Withdraw Money
* 👤 View Account Details
* ✏️ Update Account Information
* 🗑️ Delete an Account
* 💾 Store Account Data using JSON

---

## 🛠️ Tech Stack

* Python 3
* Streamlit
* JSON
* VS Code
* Git & GitHub

## 💡 Available Operations

### ➕ Create Account

* Create a new bank account
* Generates a unique account number automatically
* Initial balance is ₹0

### 💰 Deposit Money

* Deposit money into an existing account
* Balance updates instantly

### 💸 Withdraw Money

* Withdraw money securely
* Prevents withdrawal if the balance is insufficient

### 👤 Show Account Details

* View account holder information
* Display current account balance

### ✏️ Update Details

* Update name
* Update email
* Change account PIN

### 🗑 Delete Account

* Permanently delete an account after PIN verification

---

## 📁 Data Storage

The project stores all account information in a local **data.json** file.

Each account contains:

* Name
* Age
* Email
* PIN
* Account Number
* Balance
